# is_EU
Сheck that the country is a EU member state by 2 character country code (iso 3166)

## Installation
This project using composer.
```
$ composer require d-novation/is_eu
```

## Usage
Сheck that the country is a EU member state by 2 character country code (iso 3166)
```php
<?php
$result = is_EU('DE'); // true

```
